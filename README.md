## Download

Head to the [**Releases**](https://github.com/jvmisxn/jmo-vector-release-/releases/latest) page and grab the installer for your platform:

| Platform | File |
|----------|------|
| Windows  | `.msi` (recommended) or `-setup.exe` |
| macOS    | `.dmg` |
| Linux    | `.AppImage` or `.deb` |

### Verify your download (optional)

Each release includes a `checksums.sha256` file. On Windows:

```powershell
(Get-FileHash .\jmo-vector-*-windows-x86_64.msi -Algorithm SHA256).Hash
```

Compare against the matching line in `checksums.sha256`.
